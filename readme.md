# Spooky Platformer

![hackatime](https://hackatime-badge.hackclub.com/U0785D5VDEK/spooky)

A platformer game with spooky ghosts and dark lighting

Tileset from https://kenney.nl/assets/pixel-platformer and https://kenney.nl/assets/pixel-platformer-farm-expansion

Sound Effects made with sfxr

Made with the Godot Engine

## To play:

Run the web version or download the desktop buils from the game's page on [itch.io](https://xanderath.itch.io/spooky-platformer)

or

1. Download Godot from https://godotengine.org/

2. Download the repo using
``git clone https://github.com/Emarald64/spooky.git``

4. Import the folder you cloned the repo into Godot
